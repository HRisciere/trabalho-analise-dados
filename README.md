# ANÁLISE DE DADOS C3.A1
O trabalho desenvolvido visa realizar uma analise exploratória completa dos dados e aplicação de algoritmos de aprendizagem supervionada e não supervisionada.
Para realização das atividades solicitadas, foram utilizados os dados site [Kaggle](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data).

## ORGANIZAÇÃO REPOSITÓRIO

- [Data](Data): Nesse diretório estão contidos os dados utilizados para os exercícios.
  * [data_description.txt](Data/data_description.txt): arquivo .txt explicando o significado de cada uma das colunas do dataset;
  * [sample_submission.csv](Data/sample_submission.csv): arquivo .csv de submissão de referência de uma regressão linear no ano e mês da venda, metragem quadrada do lote e número de quartos;
  * [test.csv](Data/test.csv): arquivo .csv com dados para utilização no teste dos algoritmos desenvolvidos;
  * [train.csv](Data/train.csv): arquivo .csv com dados para utilização no treino dos algoritmos desenvolvidos;
- [Analise_Exploratoria_de_Dados.ipynb](Analise_Exploratoria_de_Dados.ipynb): Script Python responsável pela análise exploratória de dados, correspondente à tarefa "1.Análise exploratória de dados".
- [Aprendizagem_Nao_Supervisionada.ipynb](Aprendizagem_Nao_Supervisionada.ipynb): Script Python responsável pelos algoritmos de aprendizagem supervisionada, correspondente às tarefas "2. Featuring Engineering". e "3. Aprendizagem Supervisionada"("a. Regressão Linear","b. Classificação")
- [Aprendizagem_Supervisionada.ipynb](Aprendizagem_Supervisionada.ipynb): Script Python responsável pelos algoritmos de aprendizagem não supervisionada, correspondente à tarefa "4. Aprendizagem não supervisionada"("a. Clusterização","b. Redução de dimensionalidade","c. Análise de associação","d. Análise de outlier")
